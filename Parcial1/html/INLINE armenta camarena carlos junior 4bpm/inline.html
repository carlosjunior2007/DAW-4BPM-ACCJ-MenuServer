Primero vamos a aaprender la opcion en linea.
El css inline se utiliza pra dar estilos a un elemento HTML especifico. Para selectores.
Este tipo de CSS no es realmente recomendable, ya que cada etiqueta HTML debe ser utilizada individualmente. La gestion de tu sitio web puede resultar demasiado dificil si solo utilizas CSS inline
Sin embargo , el CSS inline en HTML puede ser util en algunas situaciones. Por ejemplo, en los casos en los que tienes acceso a archivos CSS o necesitas aplicar estilos para un solo elemento
